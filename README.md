1. Project title: Credit Card Transactions Fraud Detection
2. Project objective: Explore and analyze credit card transaction data using Pandas - Identify patterns and characteristics of fraudulent transactions - Provide actionable insights for fraud prevention
3. Tools used: Python, Pandas, Matplotlib
4. Key analyses & insights: Fraud Concentration: ~1% of transactions are fraudulent.
                            High-Risk Categories: Online shopping, gas transport, and travel.
                            Timing Patterns: Most fraud occurs late at night.
                            Customer Segments: Age 30–50 shows higher fraud risk.
                            Distance Factor: Fraud often occurs when transaction location is far from customer location.
                            Geography: Certain cities/states have higher fraud rates.
4. Dataset Source: https://www.kaggle.com/datasets/kartik2112/fraud-detection
    Note: This project performs analysis on the fraudTest.csv file from the dataset.
           A smaller sample dataset is included in the repository for demonstration purposes.
